Troubleshooting
iPhone kann nicht zugreifen
  • WLAN prüfen: Beide Geräte im gleichen Netzwerk?
  • IP-Adresse korrekt: ipconfig/ifconfig nochmals ausführen
  • Port verfügbar: Ist Port 8080 frei?
  • Firewall: Windows/macOS Firewall kann Port blockieren
  • VPN/Proxy: Temporär deaktivieren
Docker Probleme
# Ports prüfen
netstat -an | grep 8080

# Container Status
docker ps -a

# Logs anzeigen
docker logs ndcapp-preview
